# 미니 쇼핑몰 구현
<hr>

 다양한 코디와 상품을 구경할 수 있는 미니 쇼핑몰 

 - 멀티캠퍼스 백엔드 개발자 부트캠프(스프링) 미니프로젝트 3조

# 프로젝트 소개 📝
### 기획 의도 
다양한 코디와 상품을 탐색하며 원하는 상품을 검색하면서 간편 결제를 통해 빠르게 결제 할 수 있으며 
회원유저가 게시판을 통해 문의와 환불 및 교환을 할 수 있어 편리하게 홈페이지를 사용할 수 있도록 기획

## 프로젝트 기간
2024-09-26 ~ 2024-11-11
